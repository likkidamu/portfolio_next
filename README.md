# Portfolio — Damodhar Datta Reddy Likki

A personal portfolio website built with Next.js, showcasing skills, experience, and projects as a Full Stack Developer.

## Tech Stack

- **Framework:** Next.js 13 (Pages Router) + React 18
- **Styling:** Tailwind CSS 3
- **Animations:** Framer Motion
- **Sliders:** Swiper.js
- **Particles:** tsparticles
- **Email:** EmailJS
- **Icons:** react-icons

## Pages

- **Home** — Hero section with bio, avatar, and particle effects
- **About** — Skills, awards, experience, and credentials with animated counters
- **Services** — Carousel showcasing offered services
- **Work** — Project gallery with hover overlays linking to live demos
- **Contact** — Contact form powered by EmailJS
- **Testimonials** — Client testimonials

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
npm start
```

## Project Structure

```
├── components/       # Reusable UI components (Nav, Header, Layout, Sliders, etc.)
├── pages/            # Next.js pages (home, about, services, work, contact, testimonials)
├── public/           # Static assets (images, SVGs)
├── styles/           # Global CSS (Tailwind)
├── variants.js       # Framer Motion animation variants
├── tailwind.config.js
└── next.config.js
```
